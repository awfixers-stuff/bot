# Starboard

Starboard service for highlighting popular messages.

This module implements a starboard system that automatically reposts messages
to a designated channel when they receive enough reactions, allowing communities
to highlight and celebrate high-quality content.

## Classes

Discord cog for starboard functionality.

This cog monitors reactions on messages and automatically posts messages
to a designated starboard channel when they reach a configured threshold.
