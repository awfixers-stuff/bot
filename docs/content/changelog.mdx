<Update label="v0.1.0" description="Initial development release">
  Bot is now live under active development. This initial release includes core
  moderation, utility, and community features:

  - **Moderation**: Ban, tempban, kick, timeout, jail, warn, case management, purge, slowmode, report system
  - **Utility**: AFK tracking, reminders, polls, timezone conversion, code execution (eval/run), encode/decode
  - **Fun**: Random generators, xkcd integration
  - **Levels**: XP tracking and leaderboard system
  - **Snippets**: Create, edit, delete, and manage reusable message snippets
  - **Info**: User and server info, avatar, member count commands
  - **Self-hosting**: Docker Compose and systemd deployment guides
</Update>
